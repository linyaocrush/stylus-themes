# My Stylus Themes
个人制作的 Stylus 浏览器主题合集，仅提供免费个人使用，禁止未经许可修改与二次分发。

---

## 包含的主题
- SteamPeek MD3 主题 — 为 SteamPeek 打造的 Material Design 3 风格界面
（后续新增主题会持续更新）

---

## 如何使用
1.  安装浏览器扩展：Stylus
2.  打开对应主题的 `.css` / `.user.css` 文件
3.  全选代码 → 复制
4.  在 Stylus 中新建样式 → 粘贴 → 保存
5.  刷新对应页面即可生效
